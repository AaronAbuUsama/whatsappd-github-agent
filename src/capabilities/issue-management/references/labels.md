# Canonical triage labels

This is technical repository reference data, not participation policy. It mirrors the canonical triage-role mapping in `docs/agents/triage-labels.md`; confirm availability through the repository's issue options before applying a label.

| Label | Triage meaning |
| --- | --- |
| `needs-triage` | Maintainer needs to evaluate this issue |
| `needs-info` | Waiting on the reporter for more information |
| `ready-for-agent` | Ready to implement by an agent |
| `ready-for-human` | Requires human implementation |
| `wontfix` | Will not be actioned |
